Requires WooCommerce plugin to work
Plugin configurations: WooCommerce -> Woo Telegram Noticer

How to use this plugin: 
1. Go to plugin location (WooCommerce -> Woo Telegram Noticer) by default. 
2. Into api field insert your token. Telegram API token you can get from https://t.me/BotFather. 
3. Into chat id field insert your id. To get your chat id destination write this bot https://t.me/JsonDumpBot.
